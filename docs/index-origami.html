FAQ

Questions fréquentes

L'IA fonctionne avec tous les documents ?

Oui ! Notre IA reconnaît les DPE, diagnostics, factures d'énergie, manuels d'équipements, contrats... Elle extrait automatiquement les dates, numéros, caractéristiques et équipements.

Mes données sont-elles sécurisées ?

Vos documents restent sur votre téléphone. L'IA analyse localement. Rien n'est envoyé sur nos serveurs. C'est 100% privé.

C'est obligatoire pour qui ?

Pour tous les propriétaires en France. Depuis janvier 2023, vous devez remettre ce carnet à l'acheteur lors d'une vente. Pour les locations, c'est recommandé mais pas encore obligatoire.
